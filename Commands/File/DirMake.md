# DirMake

디렉터리를 생성합니다.

## 문법

```pebakery
DirMake,<DestDir>
```

### 인자

| 인자 | 설명 |
| --- | --- |
| DestDir | 생성하고자 하는 디렉터리의 전체 경로 |

## 유의사항

주의사항 없음

## 관련 명령어

[DirCopy](./DirCopy.md), [DirDelete](./DirDelete.md), [DirMove](/DirMove.md)

## 예제

### 예제 1

```pebakery
// C:\Temp가 생성됩니다.
DirMake,C:\Temp
```
